# RoofAge - Roof Aging Tool for Home Inspectors

**Date:** January 29, 2026
**Type:** Platform | **Market:** B2B | **Target:** Real Estate

## Overview

A photo-based roof age estimation tool for home inspectors. Upload a photo of roof shingles, get an AI-powered age estimate with confidence scores.

## Tech Stack

- **Framework:** Next.js 14.2.x (Vercel compatible)
- **Language:** TypeScript
- **Styling:** Tailwind CSS 3.4.x
